## 📁 文件结构
```
/var/www/gdxcsh-admin/
├── app.py              # Flask 后端（完善版）
├── index.html          # 前端主页面
├── data/
│   └── admin.db        # SQLite 数据库
└── init_db.py          # 数据库初始化脚本
```

## 🚀 部署后更新
1. 更新 app.py 到服务器
2. 运行 init_db.py 初始化数据库
3. 导入现有数据
4. 重启 Flask 服务
